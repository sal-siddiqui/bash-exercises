| Shortcut | Action                              |
| -------- | ----------------------------------- |
| Ctrl-A   | Move to start of line               |
| Ctrl-E   | Move to end of line                 |
| Ctrl-B   | Back one character                  |
| Ctrl-F   | Forward one character               |
| Alt-B    | Back one word                       |
| Alt-F    | Forward one word                    |
| Ctrl-K   | Cut to end of line                  |
| Ctrl-U   | Cut to start of line                |
| Ctrl-W   | Cut previous word                   |
| Alt-D    | Cut next word                       |
| Ctrl-Y   | Paste (yank)                        |
| Ctrl-T   | Swap last two characters            |
| Alt-T    | Swap last two words                 |
| Ctrl-P   | Previous command in history         |
| Ctrl-N   | Next command in history             |
| Ctrl-R   | Reverse search history              |
| Ctrl-G   | Cancel search / exit prompt         |
| Ctrl-C   | Interrupt running process           |
| Ctrl-Z   | Suspend running process             |
| Ctrl-D   | End of input (logout if line empty) |
| Ctrl-L   | Clear screen                        |
| Ctrl-S   | Freeze terminal output              |
| Ctrl-Q   | Unfreeze terminal output            |
